A modern personal portfolio Android application built using **Jetpack Compose**, showcasing my professional background, skills, and interests.
very simple application which has sidebar(Drawer) and Bottombar.

---

- 🧠 Tech Skills
- 💼 Work Experience
- 🎓 Education
- 📞 Contact Information
- 🏏 Hobbies
  
---

## ✨ Features

- 💡 **Modern UI** with Material 3
- 📱 Built entirely using **Jetpack Compose**
- 🎨 Clean architecture and component-based design
- 🧠 Responsive layouts for experience, skills, and education
- 📬 Copyable email & phone fields
- ⚙️ Drawer & Bottombar

---

## 🛠️ Built With

- **Kotlin**
- **Jetpack Compose**
- **Material 3 (Material You)**
- **Android Studio Giraffe+**
